# No YouTube Reels

---

## Download da Versão Mais Recente

Você pode baixar a versão mais recente da extensão diretamente do build automatizado. O arquivo `.zip` é gerado a cada nova atualização no código.

### Como baixar o arquivo:
1. Clique nesse [link](https://github.com/carvalinh0/no-youtube-shorts/actions/workflows/zip.yml) acima para ver o histórico de builds.
2. Clique no item mais recente da lista (o que estiver no topo com um ✔️ verde).
3. Na página seguinte, role para baixo até a seção **"Artifacts"** e clique em **`extension`** para baixar o arquivo `.zip`.

---
### Como Instalar (firefox)
1. Após baixar o arquivo .zip, abra a página about:addons na barra de pesquisa
2. Clique na engrenagem e em seguida em "Instalar add-on por um arquivo"
3. Adicione o zip instalado e pronto.

### Como Instalar (chrome)
1. Após baixar o arquivo .zip, abra a página chrome://extensions/ na barra de pesquisa
2. Extraia o conteúdo do zip para um local de sua escolha
3. Habilite o "modo desenvolvedor", clique em carregar sem compactação e adicione o **conteúdo do zip**.
